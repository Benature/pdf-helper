## 组合拳✊

（个人经验）

- 找到一份影印电子书，一页 PDF 内有两页书（即左页与右页），而且没目录。
- 用 [PDF 剪刀](./pdf_scissors)切割成一页 PDF 只含一页书。
- 用[批量增加书签工具](./add_bookmarks)给 PDF 打上书签🔖  
- 开始快乐高效读书
